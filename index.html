CGI — Cell Global Identifier
The CGI uniquely identifies a 2G or 3G cell globally.
Structure
Format:MCC + MNC + LAC + CI
MCC:3 digits (0–999)
MNC:2 or 3 digits (0–999)
LAC:16 bits (0–65,535)
CI:16 bits (0–65,535)
[Step1]: Select Encode or Decode mode
[Step2]: Enter the required values
[Step3]: Press "Calculate" button
Operation Guide

CGI (Cell Global Identifier) is used in 2G (GSM) and 3G (UMTS) networks to uniquely identify a cell worldwide.

Encode: Enter MCC, MNC, LAC, and CI to generate the full CGI string.

Decode: Enter a CGI string (e.g., "001-01-100-5") to parse it into its individual components with decimal and hexadecimal representations.

Example: MCC=001, MNC=01, LAC=100, CI=5 → CGI = 001-01-100-5

ECGI — E-UTRAN Cell Global Identifier
The ECGI uniquely identifies an LTE cell globally using the ECI (E-UTRAN Cell Identifier).
Structure
Format:MCC + MNC + ECI
eNodeB ID:20 bits (0–1,048,575)
Cell ID:8 bits (0–255)
ECI:28 bits (0–268,435,455)
Formula
ECI = (eNodeB_ID × 256) + Cell_ID
[Step1]: Select Encode or Decode mode
[Step2]: Enter the required values
[Step3]: Press "Calculate" button
Operation Guide

ECGI (E-UTRAN Cell Global Identifier) is used in 4G LTE networks.

ECI Structure: The 28-bit ECI is composed of a 20-bit eNodeB ID (upper bits) and an 8-bit Cell ID (lower bits). ECI = eNodeB_ID × 256 + Cell_ID.

Encode: Enter MCC, MNC, eNodeB ID, and Cell ID to compute the ECI and full ECGI.

Decode: Enter an ECI value (decimal or hex with 0x prefix) to decompose it into eNodeB ID and Cell ID.

Example: eNodeB=12345, Cell ID=2 → ECI = (12345 × 256) + 2 = 3,160,322

NCI — NR Cell Identifier
The NCI is the 5G NR equivalent of ECI, used within the NCGI (NR Cell Global Identifier).
Structure
Format:MCC + MNC + NCI
NCI:36 bits total
gNodeB ID:22–32 bits (configurable)
Cell ID:4–14 bits (= 36 − gNB bits)
Formula
NCI = gNodeB_ID × 2^(Cell_ID_bits) + Cell_ID
Default (22-bit gNB, 14-bit Cell):
NCI = gNodeB_ID × 16384 + Cell_ID
[Step1]: Select Encode or Decode mode
[Step2]: Set bit split & enter values
[Step3]: Press "Calculate" button
Operation Guide

NCI (NR Cell Identifier) is the 5G equivalent of ECI, used within the NCGI structure (NR Cell Global Identifier).

Bit Split: The 36-bit NCI is split between the gNodeB ID and Cell ID. The standard 5G macro setup uses 22-bit gNodeB ID and 14-bit Cell ID, but this is configurable by the operator.

Encode: Enter MCC, MNC, gNodeB ID, and Cell ID to compute the NCI and full NCGI.

Decode: Enter an NCI value (decimal or hex) and select the bit split to decompose it into gNodeB ID and Cell ID.

Example (22/14 split): gNodeB=100, Cell ID=1 → NCI = (100 × 16384) + 1 = 1,638,401
